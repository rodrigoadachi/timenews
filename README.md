### Instalação

yarn ou npm install

### Configuração de ambiente

- na variável de ambiente adicione as seguintes variáveis

apiBase
apiSecret

Nas variáveis:

apiBase atribuir o seguinte valor "https://newsapi.org/v2/everything"

apiSecret nescessita solicitar sua chave no endereço https://newsapi.org/account

### Inicializando a aplicação

yarn dev ou rmp run dev

- Acesse em seu navegado o seguinte endereço

http://localhost:3000/
